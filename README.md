# AB-test-statisctics
Проект по статистическому анализу данных. Состоит из двух независимых частей: разведочного анализа с проверкой гипотез на данных интернет-магазина одежды и анализа результатов A/B-теста.

## Стек

`Python` · `pandas` · `numpy` · `matplotlib` · `seaborn` · `scipy` · `statsmodels`


## Часть 1. A/B-тест — `GP_4_Statistics_AB.ipynb`

Анализ результатов A/B-тестирования новой версии лендинга в компании из сферы образования. 
Метрика — конверсия (доля зарегистрировавшихся пользователей).

**Что сделано:**

- **Проверка корректности теста:** оценка однородности групп (x^2-тест на равенство размеров), поиск дублирующихся пользователей и некорректного сплита (несовпадение `group` и `landing_page`)
- **Два сценария обработки данных:** с исключением дублирующихся пользователей и с их учётом (только первое взаимодействие + удаление некорректных записей)
- **Проверка гипотезы** о росте конверсии в экспериментальной группе через **z-тест для долей**, оценка минимального детектируемого эффекта (MDE), анализ динамики конверсии по дням и неделям
- **Дополнительно:** bootstrap-оценка разницы конверсий с доверительным интервалом, анализ доли некорректных записей и конверсии по времени суток.

## Часть 2. EDA и гипотезы — `ГП4_EDA.ipynb`

Разведочный анализ кликстрима интернет-магазина одежды (датасет *e-shop clothing 2008*, апрель–август 2008).

**Что сделано:**

- **Предобработка:** декодирование категориальных признаков в читаемые названия (страны, категории, цвета, расположение на странице), переименование колонок, сборка «каталога» товаров (`df_aggregated`), очистка от ошибочных строк.
- **EDA:** анализ структуры каталога (категории, цвета, цены, расположение, поза модели), сессий, поведения пользователей по категориям/цветам/цене/странам, активности по дням недели и переходов между карточками товаров.
- **Статистические проверки** по ходу анализа: тест Шапиро–Уилка на нормальность, x^2-тест независимости, z-тест для долей с поправкой Бонферрони, тест Манна–Уитни.
- **Три основные гипотезы:**
  - Пользователи с сериями из >= 3 кликов на одну категорию совершают в среднем больше кликов за сессию (Манна–Уитни).
  - Следующая просматриваемая категория зависит от предыдущей (x^2-тест).
  -  Пользователи из Польши кликают на более дешёвые товары, чем зарубежные (тест Левена + t-тест Уэлча, проверка Манна–Уитни).
